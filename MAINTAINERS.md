## Maintainer Responsibilities

All maintainers are expected to:

- Review and merge pull requests
- Triage issues and set priorities  
- Ensure code quality and test coverage
- Maintain backward compatibility
- Respond to security issues promptly
- Participate in project direction discussions
- Mentor new contributors
- Keep documentation up to date

See [GOVERNANCE.md](GOVERNANCE.md) for full maintainer responsibilities.

## Becoming a Maintainer

Contributors who demonstrate sustained involvement, technical expertise, and alignment with project values may be invited to become maintainers.

See [GOVERNANCE.md#becoming-a-maintainer](GOVERNANCE.md#becoming-a-maintainer) for the path to maintainership.
